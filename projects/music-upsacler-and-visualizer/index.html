Music Upsacler and Visualizer

Upscales and visualizes songs with a before and after

This was made because I already had logic to do upscaling, which was basically chaining different peoples tools to programatically do the best I can to upsacle old music from records players and vinyl. When I uploaded these initially, I just used a still image. I thought why not add some sort of visual and keep the main thing I cared about, which was the upscaling.

The audio on all of these videos is from the video on the right ("AFTER"). The only thing left behind is the visualize from the pre-processed audio (seen in "BEFORE").

The youtuber Tsoding had created a tool in C which was a GUI to open audio files and create a visualizer for them. It is very snappy and quick, and I really wanted to use it, but I have little C knowledge. I just needed to have it be so instead of opening the GUI and passing the file through there, I can pass a file path to an audio file, and get back the visualization of the audio. I forked the project and had Gemini create the simple change, which now is on my github under musializer-file-paths.
