Framework Led Matrix

Python playkit for the Framework 16 LED matrix

I, like many others who were excited about getting the Framework 16, saw that many they offered a side panel of LED matrices that they say is programmable. I got them with my order and installed them and loved the bootup sequence they play.

However, after searching online, there weren't many tools created. Framework created some rust tools, which I used to get the hex definitions of the commands to send to the boards, but in general no one had created any cool tools. Someone had created website found at ledmatrix.frame.work using javascript which allowed you to play around with your board, but nothing here saved. This really was nothing more then confirming the LEDs work. I used this javascript for help on how to send the commands to understand the logic.

I thought since this they had provided the hex codes to send commands, there should be something I could make to play cool animations on the thing I spent $100+ for.

The tool contains a detailed README with all the commands you can run, you can read more about it there, but it can plot math functions, show anagrams, play the game of life, run a traffic and particle simulator. Anything that you can do with a 2 dimensional matrix, you can do on these things. If you have a Framework 16, I really hope you give the tool a try.
