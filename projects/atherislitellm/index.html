AtherisLiteLLM

Creates Atheris hooks for python functions using any LLM allowed by LiteLLM

This project was created with the goal of finding a personal project which would fuse both Cybersecurity and AI. Since AI is the game at the moment, I felt it was crucial to bring both together in a novel and unique idea.

The tool takes prompts that you can make (or use what I have pre-placed) and creates runnable python files which fuzz 1 function out of the entire project. If you have a project with 98 functions (not including tests), then the tools output would be 98 unique python files designed to fuzz each function.

Note, the quality of the python files is determined by a lot of things. The main 2 questions to answer in order to get a good output: how good is the model you are using in terms of programming, and how good is your prompt.

I myself have not yet tested enough with different LLMs and prompts. There is research to be done using this tool regarding the outputs of differing LLMs and differing prompts.
