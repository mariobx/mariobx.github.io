Grading Analysis Tool

Analyze and visualize grading data to find patterns and insight

The Grading Analysis Tool was one of the first big projects I was a part of. I had joined a team named the Educational Data Mining team at Fordham and I was anxious to find anything to do.

I was assigned to create a GUI for some of the functions. The team had with the intentions of generalizing the tool enough such that any university can export their data as a CSV, rename key columns to our specification, and run the tool to get their own insights.

After creating most of the GUI function, no one had done any work on expanding the functions of the tool themselves or working on the logic to ensure that the tool was fast. I rewrote most of the logic using pandas and numpy, as before a lot of the calculations were hardcoded in python. I expanded and added as many calculations as were possible using the specific dataset we had, which was not a lot, as Fordham did not give us as much data as we were hoping for in the quality we were hoping for.

The tool is capable of many groupings and subgroupings. It can create all sorts of graphs based on the users choice. A key idea was that universities can use these in reports, so a lot of effort was placed to be able to customize the visual look of the graphs as much as possible such that whatever theme matplotlib supported could be used.

The structure of the tool is not proper for a python project like this, and it is now slightly confusing to traverse it. What can you do, it was my very first big project. The next steps in this project would be to restructure with code base to make it easier to expand. It would not be too hard, however to my knowledge this project is dead.
